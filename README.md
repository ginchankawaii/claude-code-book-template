# 💗 ENCOR合格ストーリー 〜あかりと目指す9月合格〜

CCNP ENCOR(350-401)に**2026年9月(上期末)合格**するための、ギャルゲー風・学習継続システムです。
毎日の勉強を記録すると後輩・**橘あかり**の好感度が上がり、レベルアップでイベントが発生。合格報告でハッピーエンドを迎えます。

## 使い方

1. **`index.html` をブラウザで開く**(ダブルクリックするだけ。インストール不要)
   - スマホで使う場合はこのリポジトリをGitHub Pagesで公開するか、ファイルをスマホに送って開く
2. 勉強したら**その日のうちに記録**(分野・問題数・正答率・時間・メモ)— 1分で終わります
3. あかりちゃんが褒めてくれる → 好感度アップ → 続けたくなる → 合格

データはブラウザのlocalStorageに保存されます。月1回「データ書き出し」でバックアップ推奨。

## ゲームシステム

| 要素 | 内容 |
|---|---|
| 好感度 (0〜100) | 毎日の記録で上昇。30分以上・40問以上・正答率80%以上・7日連続でボーナス |
| レベル | Lv.1 顔見知り → Lv.2 勉強仲間 → Lv.3 気になる先輩 → Lv.4 特別な存在 → Lv.5 両想い(まであと一歩) |
| イベント | レベルアップ時・連続7/14/30日達成時に特別イベント発生 |
| 復帰歓迎 | 3日以上サボっても怒られません。おかえりイベントで再スタート |
| ハッピーエンド | 「合格しました!!」ボタンで告白エンド 🌸(下期ENARSI編に続く) |
