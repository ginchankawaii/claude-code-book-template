# 🧪 ENCOR合格ストーリー 〜タキと目指す9月合格〜

CCNP ENCOR(350-401)に**2026年9月(上期末)合格**するための、ギャルゲー風・学習継続アプリです。
毎日の勉強を記録すると、検証ラボの同期・**早瀬タキ**(ツンデレ天才研究者)の好感度が上がり、レベルアップでイベントが発生。合格報告でハッピーエンドを迎えます。

> タキ『べ、別にキミのためにこのアプリがあるわけじゃないんだからね。……データが欲しいだけさ』

## 📱 iPhoneにアプリとして入れる方法(PWA)

このアプリはPWA対応済みなので、ホーム画面に追加すると**普通のアプリと同じように全画面・オフラインで動きます**。

1. このリポジトリをGitHub Pagesで公開する(初回のみ)
   - GitHubの **Settings → Pages → Branch: main → Save**
   - 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` でアクセスできるようになる
2. iPhoneの**Safari**でそのURLを開く
3. **共有ボタン(□↑) → 「ホーム画面に追加」**
4. ホーム画面にタキのアイコンが追加される → 以降はアプリとして起動🧪

データはブラウザ(ホーム画面アプリ)内に保存されます。月1回「データ書き出し」でJSONバックアップ推奨。

※PCで使う場合は `index.html` をダブルクリックするだけでも動きます。

## 🎮 ゲームシステム

| 要素 | 内容 |
|---|---|
| 好感度 (0〜100) | 毎日の記録で上昇。30分以上・40問以上・正答率80%以上・7日連続でボーナス |
| レベル | Lv.1 観察対象 → Lv.2 共同研究者 → Lv.3 興味深いキミ → Lv.4 かけがえのない相棒 → Lv.5 運命共同体(たぶん) |
| イベント | レベルアップ時・連続7/14/30日達成時に特別イベント発生 |
| 表情 | ニヤリ(通常)・デレ・キラキラ・ツン(サボると拗ねる)の4表情 |
| 復帰歓迎 | 3日以上サボっても「心配なんかしてない」と言いながら迎えてくれます |
| ハッピーエンド | 「合格しました!!」ボタンでツンデレ告白エンド 🌸(下期ENARSI編に続く) |

## 👩‍🔬 早瀬タキ(はやせ・タキ)
