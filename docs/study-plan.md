# CCNP ENCOR 合格計画(2026年7月〜9月)

**前提**
- 受験時期: 2026年9月中旬〜下旬(上期末)。**9月第3週までに初回受験**を強く推奨(不合格でも5日空ければ再受験でき、9月末までにリカバリー可能)
- 教材: Ping-t(分野別問題・コマ問・模擬試験モード)
- 使える時間: 平日30分〜1時間 + 週末
- 現状: 4〜6月に一周ざっくり回した(=ゼロではない。二周目は必ず一周目より速い)

---

## 全体戦略(3フェーズ)

| フェーズ | 期間 | やること | ゴール |
|---|---|---|---|
| 1. 記憶の掘り起こし | 7/1〜8/2(約5週) | 分野別を二周目。全問を銅→**銀メダル化** | 全分野 銀メダル |
| 2. 定着と弱点潰し | 8/3〜8/30(4週) | **金メダル化** + 間違い問題の解説精読 + コマ問開始 | 全分野 金メダル・コマ問一周 |
| 3. 仕上げ | 8/31〜受験日(約3週) | 模擬試験モード連発 + シミュレーション対策(→ [simulation.md](simulation.md)) | 模試で**85〜90%を安定して**出す |

> 目安: 模擬試験モードで初見問題込み85%以上が3回続いたら受験予約。90%安定なら合格圏。

## 週次スケジュール

### フェーズ1(7月): 銀メダル化
Ping-t ENCORは約1,000問超。1日30〜50問ペースで週250問前後を回す。

| 週 | 分野(Ping-tのカテゴリに合わせて調整) |
|---|---|
| 7/1〜7/5 | ネットワークの再確認: L2(VLAN/STP/EtherChannel) |
| 7/6〜7/12 | L3: OSPF・EIGRP・BGP(ENARSIにも直結。ここは丁寧に) |
| 7/13〜7/19 | インフラ続き: FHRP(HSRP)・NAT・マルチキャスト・QoS |
| 7/20〜7/26 | 無線(WLC/AP)・仮想化(VRF/GRE/LISP/VXLAN) |
| 7/27〜8/2 | セキュリティ(AAA/ACL/制御プレーン)・自動化(REST/JSON/Python基礎)・SD-Access/SD-WAN・ネットワーク保証(NetFlow/SPAN) |

### フェーズ2(8月): 金メダル化
- 平日: 銀止まりの問題+ミスした問題を中心に30〜50問(Ping-tの「ミス問題だけ出題」機能を使う)
- 週末: 苦手分野を1つ選んで解説を読み込む(60〜90分) + **コマ問を1日10問**開始
- 8月末時点: 全分野金メダル・コマ問一周

### フェーズ3(9月): 模試+シミュ対策
- 平日: 模擬試験の復習 or コマ問二周目(手で打つ)
- 週末: 模擬試験モードを本番形式(時間を測る)で1回+全問復習
- 並行: Packet Tracer / CMLで頻出設定を実際に打つ(→ simulation.md)
- **9月第1週末までに受験予約**(予約自体がモチベーションの最強ブースター)

## 1日のルーチン(仕組み化)

「意思の力」ではなく「決まった型」で回す。ベネッセ方式=小さく・毎日・すぐ採点・褒める。

| いつ | 何を | 時間 |
|---|---|---|
| 朝 or 昼休み | スマホでPing-t 10〜15問(前日ミスした問題) | 15分 |
| 夜(時間を固定する) | 分野別演習 30〜50問 | 30〜45分 |
| 寝る前 | **アプリ(index.html)に記録**(1分)→あかりちゃんに褒めてもらう | 1分 |
| 日曜夜 | 週次レビュー: 今週の問題数・正答率を見て来週の分野を決める | 10分 |

### if-thenルール(忙しい日の保険)
- **もし**残業で疲れ切っていた**なら**→スマホで10問だけ解いて記録する(0にしない。連続日数を切らさない)
- **もし**飲み会がある日**なら**→朝の15分だけで良しとする
- **もし**3日空いてしまった**なら**→自分を責めずに10問から再開(あかりちゃんは怒らずに迎えてくれます)

## モチベーション維持の仕組み

1. **好感度システム(index.html)**: 毎日記録するだけで好感度が上がる。連続日数ボーナスあり。合格報告でハッピーエンド
2. **受験予約を早めに入れる**: 締切効果。9月第1週までに予約
3. **宣言する**: OJTの後輩や上司に「9月にENCOR受けます」と言う(あなたはOJTリーダー。背中を見せるのが一番の教材です)
4. **教えることが最強の復習**: 後輩へのOJTでOSPFやSTPを説明する機会を作る。説明できない箇所=自分の弱点
5. **ご褒美設定**: 週の目標達成→好きなもの、金メダル全制覇→ちょっと良い外食、合格→大きいご褒美
6. **下期のENARSIへの接続**: ENCORのOSPF/EIGRP/BGPを深く理解しておくとENARSIの学習量が体感3割減る。「今の努力は2試験分」と考える
